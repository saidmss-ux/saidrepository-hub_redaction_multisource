# DocuHub — Content Extraction & Authoring Platform

Centralized interface for content creators to import sources, extract content, and write with AI assistance.

## Features

- **Multi-format import** — PDF, DOCX, TXT files or URLs
- **Intelligent extraction** — Text content with formatting preservation
- **Parallel view** — Side-by-side source and editor
- **Video transcription** — Convert audio/video to text
- **AI assistance** — Optional AI-powered writing helpers via API key

## Prerequisites

- Python >= 3.10
- Node.js >= 18 (for frontend, when ready)
- pip (Python package manager)

## Quick Start

### Backend Development

1. **Install dependencies:**
   ```bash
   npm run install-backend
   # or manually: pip install -r requirements.txt
   ```

2. **Start dev server:**
   ```bash
   npm run dev
   ```
   This starts FastAPI on `http://localhost:3000`

3. **View API documentation:**
   Open `http://localhost:3000/docs` in your browser (auto-generated by FastAPI)

### Running Tests

```bash
# Install backend dependencies first
npm run install-backend

# Run tests
python -m pytest tests/ -v
```

## Architecture

### Backend (FastAPI)

Handles all file operations, extraction, and API proxying:
- `backend/main.py` — FastAPI application and endpoints
- `backend/models.py` — Pydantic request/response contracts
- `tests/test_api.py` — API endpoint tests

Key endpoints:
- `POST /upload` — Upload file content
- `POST /download-from-url` — Fetch content from URL
- `POST /extract` — Extract text from source
- `GET /sources` — List all uploaded sources
- `GET /source/{file_id}` — Get source details
- `POST /video-to-text` — Transcribe video (MVP)
- `POST /ai-assist` — AI writing assistance
- `GET /health` — Health check
